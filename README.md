# Калькулятор реструктуризации кредита
v2
Веб-приложение для расчета условий реструктуризации кредита, разработанное на Next.js с использованием TypeScript и SCSS.

https://github.com/evgtretyakov/credit_calc.git

## 📋 Описание проекта

Приложение позволяет сотрудникам банка (СберБанк) рассчитывать варианты реструктуризации кредита для клиентов. Калькулятор учитывает различные параметры кредита и условия реструктуризации, предоставляя детальный график платежей.

### Основные возможности

- **Расчет графика платежей** с учетом реструктуризации
- **Настройка параметров кредита**: сумма долга, процентная ставка, срок, неустойки
- **Параметры реструктуризации**: отсрочка платежей, минимальный размер погашения
- **Льготные условия**: снижение ставки при переводе зарплаты в Сбер
- **Учет исполнительных производств**: разделение дохода пополам при наличии производств у пристава
- **Генерация заявления** на реструктуризацию
- **Визуализация результатов** в табличной форме

## 🚀 Технологии

- **Next.js 14** - React фреймворк для серверного рендеринга
- **TypeScript** - статическая типизация
- **SCSS/SASS** - стилизация с поддержкой переменных и модулей
- **React 18** - библиотека для построения пользовательского интерфейса

## 📊 Использование калькулятора

### 1. Ввод данных по кредиту
- **Сумма основного долга** - текущая задолженность по кредиту
- **% ставка по кредиту** - годовая процентная ставка
- **Сумма неустоек** - начисленные пени и штрафы
- **Оставшийся срок** - количество месяцев до окончания кредитного договора

### 2. Настройка реструктуризации
- **Отсрочка (мес.)** - количество месяцев, в течение которых выплачиваются только проценты
- **Минимальный размер погашения (%)** - минимальный процент от остатка долга для ежемесячного погашения
- **Перевести ЗП в Сбер** - снижение ставки на 0.5% годовых на льготный период
- **Есть возбужденные производства у пристава** - учет разделения дохода пополам

### 3. Результаты расчета
Калькулятор отображает:
- Детальный график платежей на 12 месяцев
- Разделение платежа на проценты и основной долг
- Остаток задолженности после каждого платежа
- Общую сумму платежа с учетом неустоек

## 📝 Особенности реализации

### Алгоритм расчета
1. Расчет ежемесячной процентной ставки из годовой
2. Определение общего срока кредита (оставшийся срок + период отсрочки)
3. Для периода отсрочки: выплачиваются только проценты
4. После отсрочки: равномерное погашение основного долга
5. Применение минимального процента погашения, если установлен
6. Распределение неустоек равномерно на весь срок

### Компоненты
- **Calculator** - основной компонент, управляющий состоянием и расчетами
- **InputSection** - форма ввода параметров кредита и реструктуризации
- **ResultsTable** - таблица с графиком платежей
- **Assistant** - компонент с рекомендациями и анализом платежеспособности
- **StatementModal** - модальное окно для генерации заявления
